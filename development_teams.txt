
Запуск сервера:
python manage.py runserver
=========================

Запуск Celery на Windows
celery -A moloshop.config worker -l info -P solo
=========================

Создание миграций новых моделей:
python manage.py makemigrations

Миграции:
python manage.py migrate
=========================

список всех зависимостей в виртуальном окружении:
    создать:
pip freeze > requirements.txt
    установить:
pip install -r requirements.txt
